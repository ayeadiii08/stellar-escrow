# Stellar Escrow dApp 🚀

A decentralized Escrow application built on the Stellar Soroban Smart Contract platform.

## Features

- Connect Freighter Wallet
- Create Escrow
- Lock Funds
- Read Escrow Details
- Smart Contract Integration
- Stellar Testnet Deployment
- Transaction Status Tracking

## Tech Stack

- React
- TypeScript
- Vite
- Stellar SDK
- Soroban RPC
- Freighter Wallet
